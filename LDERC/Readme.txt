Local Dollar Exchange Rate Checker

English

Important technical information:

1. Where the exchange rate comes from
- The app gets live USD exchange rates from this public API:
  https://open.er-api.com/v6/latest/USD
- The app refreshes the rates automatically every 60 seconds.
- The app also lets the user refresh rates manually with the Refresh button.

2. What data is read from the user's device
- The app reads only the text that the user types into the app itself:
  - the USD amount in the calculator
  - the currency search text
- The app does not read personal files, documents, photos, browser data, passwords, or system settings.
- The app does not scan folders or collect hidden background data from the device.

3. What data is sent over the internet
- The app sends one request to the exchange-rate API to download public currency rates.
- The calculator amount and search text are not sent to the API.
- If the GitHub link is clicked, the default browser opens the public GitHub page:
  https://github.com/kirpi4blin-GD

4. Safety
- The app is generally safe to use for normal currency checking.
- It does not execute user commands.
- It does not give remote access to the computer.
- It does not store user personal data inside the app.
- It does depend on an external exchange-rate service, so internet availability and the accuracy of that service affect the displayed rates.


Русский

Важная техническая информация:

1. Откуда берется курс валют
- Приложение получает актуальные курсы USD из этого публичного API:
  https://open.er-api.com/v6/latest/USD
- Приложение автоматически обновляет курсы каждые 60 секунд.
- Также пользователь может обновить курсы вручную кнопкой Refresh.

2. Какие данные читаются с устройства пользователя
- Приложение читает только тот текст, который пользователь вводит в самом приложении:
  - сумму в долларах США в калькуляторе
  - текст поиска валюты
- Приложение не читает личные файлы, документы, фотографии, данные браузера, пароли или системные настройки.
- Приложение не сканирует папки и не собирает скрытые фоновые данные с устройства.

3. Какие данные отправляются через интернет
- Приложение отправляет один запрос к API курсов валют, чтобы получить публичные данные о курсах.
- Сумма калькулятора и текст поиска не отправляются в API.
- Если нажать на ссылку GitHub, откроется браузер по умолчанию и публичная страница GitHub:
  https://github.com/kirpi4blin-GD

4. Безопасность
- В целом приложение безопасно для обычной проверки курсов валют.
- Оно не выполняет пользовательские команды.
- Оно не предоставляет удаленный доступ к компьютеру.
- Оно не хранит личные данные пользователя внутри приложения.
- Приложение зависит от внешнего сервиса курсов валют, поэтому на отображаемые курсы влияют доступность интернета и точность этого сервиса.
